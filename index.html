Bienvenue sur la page de SCPScanner

Nous vous présentons un outil réseau pour scanner les ports ouverts sur un hôte.

Comment l'utiliser

Sur un terminal ou un prompt shel cmd.exe, exécutez le script comme suit:

syscore$ python SCPScanner.py -H 192.168.1.137 -p 21, 22, 80, 1720
